# Changelog

## 0.1.3

- Added an evidence gate for non-trivial Unreal Engine work.
- Required an ownership map covering entry, runtime owner, authoritative data, lifecycle, authority, persistence, C++/Blueprint handoff, and cleanup before complete implementation guidance.
- Added a collaboration checkpoint for cross-system guided implementation: evidence and the smallest design come before transcription-ready code.
- Added maintainability checks for one source of truth, current callers, empty lifecycle overrides, speculative abstractions, the change test, the deletion test, and the no-chat test.
- Added recovery guidance to retrace architecture after repeated compile, link, reflection, asset, or configuration failures instead of stacking local patches.
- Applied the new gates to planning, drafting, implementation, investigation, review, and Blueprint workflows.
